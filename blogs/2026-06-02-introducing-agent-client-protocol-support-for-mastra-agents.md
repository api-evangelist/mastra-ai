Mastra agents now support the Agent Client Protocol (ACP), enabling delegation of specialized coding tasks to Claude Agent, Codex CLI, Cursor, and compatible harnesses. The integration allows developers to offload tasks that would otherwise require custom development, expanding the range of multi-agent workflows Mastra can coordinate.
