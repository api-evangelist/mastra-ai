Mastra shipped Agent Signals, a new context engineering primitive that lets developers steer, wake, or connect multiple clients to a single running agent stream and update state without invalidating the prompt cache. Signals decouple stream ownership from context delivery, enabling more flexible multi-client agent interactions.
